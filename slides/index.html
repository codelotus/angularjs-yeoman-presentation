Modules

Should all be configured in app.js 

                    
            var app = angular.module('yourAppName', ['yourAppDep']);
            angular.module('yourAppDep');

Define Controllers on modules 

                    
            app.controller("MyCtrl", function() {});

Directives

Use an app specific prefix 

                      
            angular.module('yourAppDep')
                .directive('yourAppControlPanel', function () {
                  // ...
                });

Side note on $digest

Filters

  • Called twice for each digest cycle
  • If data will not be modified by the filter, call the filter in the controller
                              
            {{myData.author | myFilter}}
                          
                          
                              
            myApp.controller('MyCtrl', function($scop, $filter){
              $scope.myData = { title: "ABC", author: "xyz" }
              $scope.myData.author = $filter('myFilter')($scope.myData.author)
            });
